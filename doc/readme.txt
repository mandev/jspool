jSpool README

****************************************************************************
* About jSpool
****************************************************************************

jSpool is a cross platform file spooler written in Java.
jSpool was succesfully tested with SUN JRE/JDK1.6+ on WIN2K & Linux

****************************************************************************
* Installation
****************************************************************************

Windows : You can either start jSpool by double-clicking on jSpool.jar
Un*x : create a "jSpool.sh" shell script with the required paths.

ie :  "$JAVA_HOME/bin/java -jar jSpool.jar"
